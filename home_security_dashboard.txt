Home Security Dashboard
Components:

Sharp 2D120x infrared distance sensor
ESP32 Wi-Fi module + bluetooth module 
NeoPixel LED Strips

Description:
Design a security system that monitors movement and alerts users:
Use the Sharp 2D120x to detect movement or intrusion.
The device count five second (using the internal timer in the esp32)  
If no pin introduced then 
	Flash NeoPixel LEDs to indicate breaches visually.
If the pin introduced 
	then the device will be set to the sleeping mode 

ESP32 sends alerts to a mobile app or email + the ability of the home owner to stop the alarm (open the camera if it is added) 


Optional integration with cameras for enhanced surveillance is possible as well
